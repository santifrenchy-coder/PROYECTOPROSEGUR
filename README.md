# 🛡️ EXPLORADOR PRO | Movistar Prosegur Alarmas

> **Herramienta táctica de alta precisión para la prospección comercial en campo, diseñada exclusivamente para agentes de Movistar Prosegur Alarmas.**

---

## 🚀 Descripción del Proyecto

**EXPLORADOR PRO** es una aplicación web de alto rendimiento optimizada para dispositivos móviles que permite a los agentes comerciales identificar, gestionar y sincronizar oportunidades de negocio en tiempo real. 

Utilizando un sistema **"Offline-First"**, la aplicación garantiza que el agente pueda seguir trabajando en zonas con baja conectividad, sincronizando automáticamente los datos con la nube de **Supabase** cuando la conexión se restablece.

---

## 💎 Características Principales

- **🛰️ Escáner Táctico de Zona**: Localización automática de negocios mediante coordenadas GPS y visualización en radar.
- **🗺️ Mapas Interactivos**: Integración con Leaflet y Overpass API para detectar perímetros comerciales y puntos de interés en tiempo real.
- **☁️ Sincronización Cloud (Supabase)**: Backup automático y recuperación de datos multicliente para evitar la pérdida de información en campo.
- **📊 Gestión de Leads 360**: Sistema de fichas detalladas con métricas de interés, estados de gestión (Visita, Seguimiento, Firmado) y análisis de competencia.
- **📱 Interfaz Premium Dark**: Diseño ultra-moderno basado en el Design System de Prosegur, optimizado para usabilidad móvil bajo luz solar.
- **📥 Herramientas de Exportación**: Exportación instantánea a Excel y sincronización directa con bases de datos externas (Airtable).

---

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS3 (Custom Properties & Glassmorphism).
- **Mapas**: Leaflet.js + Nominatim Geocoding API.
- **Backend**: Supabase (PostgreSQL + Auth + Realtime).
- **Iconografía**: Lucide Icons.
- **Utilidades**: 
  - `XLSX.js` para exportación de datos.
  - `LZ-String` para compresión de backups locales.
  - `Radial Gradients` & `Keyframe Animations` para el sistema de radar.
